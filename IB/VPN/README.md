# Домашнее задание к занятию "Виртуальные частные сети VPN`

### Задание 1
Описание задачи.
В рамках данного задания вы получите практический опыт использования самого популярного Open Source решения для организации VPN — OpenVPN.  
Ожидаемый результат от выполнения задания:
Между двумя хостами установлено VPN-соединение, данные передаются в зашифрованном виде.

Что необходимо прислать в качестве ответа на задание:
Выполните задание, сделайте необходимые скриншоты, загрузите их на Яндекс диск и вставьте ссылку на вашу работу в поле «Ответ»:

Пришлите скриншот Wireshark, где видно, что данные передаются в открытом виде (для раздела PlainText).  
Пришлите скриншот Wireshark, где видно, что данные не передаются в открытом виде (для раздела Shared Key).
